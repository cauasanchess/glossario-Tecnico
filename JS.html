If

permite executar um bloco de código somente se uma determinada condição for verdadeira.

Else

Define o que acontece quando a condição do código "if" é falsa.

Else if

Usado para verificar várias condições antes de chegar ao "else".

Switch

É usado para executar diferentes blocos de código com base em diferentes valores de uma expressão

Case

Ele define uma condição específica para comparar o valor da variável do "switch".

Break

O break é importante para sair do switch após encontrar uma correspondência.

Default

Se nenhuma correspondência for encontrada, o código dentro do default é executado.
