BR

O br insere uma quebra de linha dentro de um parágrafo ou texto. Ele permite separar conteúdo sem criar um novo bloco, mantendo o fluxo do texto.

HR

O hr cria uma linha horizontal para separar visualmente seções ou tópicos da página. É útil para organizar conteúdo e indicar mudanças de assunto.

Video

Insere vídeos na página, com suporte a controles (play, pause, volume).

Iframe

Permite incorporar outra página ou conteúdo externo dentro do site.

Em

Serve para deixar o texto em itálico.

Strong

Serve para deixar o texto em negrito.

Select

Cria uma lista suspensa, permitindo que o usuário escolha um valor entre várias opções.

Ul

define uma lista não ordenada, onde os itens não seguem uma sequência numérica ou alfabética.

Ol

Define uma lista ordenada (numerada).

Audio

Utilizado para inserir arquivos de audio ao site.
