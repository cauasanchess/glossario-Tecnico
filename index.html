O HTML (HyperText Markup Language) e uma linguaguem de programação fundamental para a criação de páginas web .

O CSS (Cascading Style Sheets) e uma linguaguem de programação usada para estilizar e customizar os documentos HTML.

O JS (JavaScript) é uma linguagem de programação usada para adicionar interatividade e conteúdo dinâmico a páginas web.
